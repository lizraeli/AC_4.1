# CSS: Stylish Table

Source: [CSS Menu Maker](http://cssmenumaker.com/blog/stylish-css-tables-tutorial/)

## Instructions

Use the following html:

```html
<table>
<tr>
    <th>Pets</th>
    <th>Fruit</th>
    <th>Vegetables</th>
</tr>
<tr>
    <td>Dogs</td>
    <td>Cats</td>
    <td>Parrots</td>
</tr>
<tr>
    <td>Oranges</td>
    <td>Apples</td>
    <td>Pears</td>
</tr>
<tr>
    <td>Tomatoes</td>
    <td>Cucumbers</td>
    <td>Lettuce</td>
</tr>
</table>
```
